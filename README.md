# 📖 [RealTime Search] <a name="about-project"></a>

**[RealTime Search]()** is a project that allows users to search for articles in real time. It is built with Ruby on Rails and uses the searchkick gem to implement the search functionality. The searchkick gem uses elastic search to index the articles and make them searchable. The searchkick gem also allows for the use of fuzzy search which allows for the search to return results even if the search query is misspelled. The searchkick gem also allows for the use of partial search which allows for the search to return results even if the search query is a partial match of the article title.
